 HealthHub CM 

HealthHub CM is a mobile application designed to improve access to medicines by connecting users with nearby pharmacies. The app aims to solve common challenges such as difficulty in finding available medicines, long waiting times, and lack of trust in pharmaceutical services.

---

 Problem Statement

Many people struggle to:
- Know where medicines are available
- Compare prices across pharmacies
- Trust the authenticity of drugs
- Avoid long queues and unnecessary movement

Most users still rely on physically visiting pharmacies, which is inefficient and time-consuming.

---

 User Research

A survey was conducted with **10 respondents**:
- 8 Customers (medicine buyers)
- 1 Pharmacy owner
- 1 Pharmacy staff

### Key Findings:
- Most users still visit pharmacies physically
- The biggest issue is **not knowing medicine availability**
- Users want:
  - Medicine availability search
  - Nearby pharmacy listing
  - Communication with pharmacists
- Trust and verification are very important

---

 Project Goal

To build a user-centered application that:
- Helps users quickly find available medicines
- Connects users to nearby pharmacies
- Improves transparency and trust in pharmaceutical services

 Core Features (MVP)

- Search for medicine availability
- View nearby pharmacies
- Display medicine prices
- Basic pharmacy verification

Future Features

- Chat with pharmacists
- Medicine delivery
- Notifications for availability
- Medication reminders
- Loyalty and discount system

Technologies Used

- Kotlin
- Android SDK
- Jetpack Compose / ViewBinding
- Gradle

Target Users

- Customers (medicine buyers)
- Pharmacy owners and staff
 Project Status

In development — core features are being built based on user research.

Note

This project is developed for academic purposes and follows a user-centered design approach based on real survey feedback.
